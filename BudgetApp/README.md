BudgetApp

📌 Description

BudgetApp is a simple console-based prototype for managing a personal budget.
The system allows tracking incomes and expenses by categories and generating analytical reports.

This project was developed as a coursework for the discipline “Software Engineering” using test-driven development principles.

---

🎯 Functional Features

- Add income transactions
- Add expense transactions
- Categorize incomes and expenses
- Calculate total monthly income
- Calculate total monthly expenses
- Generate monthly expense reports by category
- Calculate average expenses per category

---

🛠 Technologies Used

- Programming Language: C#
- Platform: .NET
- Testing Framework: NUnit
- Version Control: GitHub
- Application Type: Console Application (logic-focused)

---

🧪 Testing

The project follows a test-oriented development (TDD) approach.

- Unit tests are implemented using NUnit
- Core business logic is fully covered by unit tests
- Test results confirm correctness of calculations and reports

To run tests:

- Open the solution in Visual Studio
- Go to Test → Run All Tests

---

📂 Project Structure

- "BudgetApp" – main application logic
- "BudgetApp.Models" – domain models (Category, Transaction, BudgetManager)
- "BudgetApp.Tests" – unit tests for core components

---

🚀 How to Run

1. Open "BudgetApp.sln" in Visual Studio
2. Build the solution
3. Run unit tests to verify system functionality
